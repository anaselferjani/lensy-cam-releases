Turn your everyday webcam into a creative AR camera. Apply fun Snapchat-style lenses live, capture photos, and stream your filtered feed directly into Zoom, Teams, OBS, or any other app via the built-in virtual camera.

## ✨ Features

- 🎭 **Live AR Lenses** — real-time Snapchat-style filters powered by Snap Camera Kit
- 📷 **Photo Capture** — snap and save high-quality photos with your active filter applied
- 🎥 **Virtual Camera** — stream your filtered feed into Zoom, Microsoft Teams, OBS, Discord, and more
- 🔄 **Mirror & Orientation Controls** — flip horizontally and switch between landscape/portrait on the fly
- ⭐ **Favorites** — save your favorite lenses for quick access
- 🔁 **Automatic Updates** — the installed (Setup) version updates itself — no manual downloads needed
- 🖥️ **Portable Option** — no installation required if you prefer a single standalone file
